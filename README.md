# Shivam Cycle Mart website

This is a static website ready for GitHub Pages. No build step or dependencies are needed.

## Deploy to GitHub Pages

1. Create a new GitHub repository (for example, `shivam-cycle-mart`).
2. Upload every file and folder from this project, including `assets/hero-frames`.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and the `/ (root)` folder, then save.

GitHub will publish the site at `https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`.
